# Streamlit App Template

This repository is a **minimal multi-page Streamlit app** template.

It shows:

- How to structure a simple Streamlit project
- How to run the app locally
- How to add additional pages

---

## 1. Prerequisites

You need:

- **Python 3.9+**
- **pip** (Python package manager)

Check in a terminal:

```bash
python --version
pip --version
